## What's worth the GG7 treatment, mapped to the systems we've actually built.

## 0. How to read this

Each candidate is rated by **ROI for a live system**, not by how good the book is. The GG7
job proved the pattern: OCR → paraphrase → **era-translate to ~20 BBY** → reorganize as design
notes → prune the scan. Two rules gate everything:

- **Era translation.** We're Clone Wars (~20 BBY). OT/Imperial-era *setting* is off-era and
  gets stripped or recast; era-agnostic *mechanics/gear/species/creatures/structure* are kept.
- **Q1 canon-character policy.** No canonical figure becomes a named NPC. Extractions reduce
  them to archetypes (as the GG7 doc did).

**WEG (D6) vs WotC (d20):** WEG books share our ruleset — extract content *and* stats. **WotC
books use d20 (DCs, feats, levels) — incompatible. Extract setting/lore/locations/NPC concepts
ONLY; re-stat to D6 from scratch or not at all.** WotC's value is that it has the CW-era setting
material WEG never wrote.

**Already extracted (don't redo):** GG6 Tramp Freighters, GG7 Mos Eisley, GG10 Bounty Hunters,
Cracken's Rebel Field Guide, Tales of the Jedi, JAS, CWCG. Plus existing Coruscant-underworld
and From-Dust-to-Stars content.

## 4. For the (deterministic) Director — the key insight

You flagged the AI director as desirable but API-expensive. Reframe: **you don't need live LLM
calls to get good direction if you mine the right books into a deterministic content library.**

- **Adventure modules ARE deterministic quest content.** Every WEG adventure is a pre-authored,
  branching structure — objectives, NPC motivations, encounter beats, complications, rewards.
  Extracted and era-translated, they become a **library of quest templates** the director
  sequences with rules, not generation. Highest-value, on-theme picks: **Tatooine Manhunt**,
  **Scavenger Hunt**, **Black Sands of Socorro**, **The Far Orbit Project**, **Death in the
  Undercity**, plus the seed hooks in **GG9: Fragments** and **Wretched Hive**.
- **GM guides give the director's heuristics.** **Gamemaster's Guide** (WEG) and the WotC
  **Galactic Campaign Guide** codify pacing, random complication/event tables, reaction rules,
  and encounter scaling — exactly the deterministic decision logic a rule-based director runs
  (when to escalate, what complication to inject, how to react to player state). Extract these
  as **director rule tables**, not in-game lore.
- **Hybrid path:** keep Haiku (or local Mistral) for *flavor text only* — the cheap, optional
  top layer — while the deterministic core (template selection + heuristic tables from the GM
  guides) drives *what happens*. That caps API cost to occasional prose, not decision-making.

Suggested first director-feed extraction set: **Gamemaster's Guide + Tatooine Manhunt + GG9**.
That's enough to stand up a deterministic quest/complication library scoped to the starting zone.

## 6. Suggested sequencing

1. **Crime + place core:** GG11 Criminal Organizations → Wretched Hive → Hideouts and Strongholds.
2. **Economy/space depth:** Platt's Smugglers Guide → Stock Ships → Pirates and Privateers.
3. **Encounters/wilderness:** Creatures of the Galaxy → GG8 Scouts.
4. **Gear:** Fantastic Technology: Guns and Gear (then Droids/Personal Gear as needed).
5. **CW-era setting (WotC, lore-only):** Secrets of Tatooine → Geonosis and the Outer Rim Worlds → Coruscant and the Core Worlds.
6. **Director + quests:** Gamemaster's Guide → Tatooine Manhunt → GG9: Fragments, then expand the quest-template library from the Tier-2/3 adventures.

Each follows the GG7 loop: OCR → paraphrase → era-translate → reorganize as design notes →
prune the scan. Net effect: every scan you extract is a scan you can then delete.
