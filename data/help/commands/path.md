Commit to your Force journey after completing all five Village trials
(Step 10 of the Village quest path).

**Syntax:**

    path             — show the path menu
    path a           — choose Path A
    path b           — choose Path B
    path c           — choose Path C (conditional)

**The three paths:**

**Path A — The Jedi Order:** You are escorted by convoy to Coruscant
Temple. The Order's intake liaison receives you. You enter the Jedi
structured path (Padawan-Master lineage system).

**Path B — The Village:** You remain with the village as an
independent Force-sensitive. The Force is yours to develop without
the Order's oversight.

**Path C — Dark whispers:** Only available if the Spirit trial locked
this route. The pull of the dark side is strong. (Irreversible.)

**This choice is final.** Once you commit, the path is set for your
character's arc. Use `path` (no argument) first to see which paths
are open to you.

**See also:** `+trial` for the full trial system; `accuse` and `gate`
for earlier trial steps; `+powers` to develop Force abilities after
committing.
