Initiate a conversation with an NPC. Responses are generated by
the Ollama NPC dialogue AI and reflect each NPC's personality,
faction, and current world events.

SYNTAX

  talk                       — list NPCs in the current room
  talk <npc>                 — start a conversation (default greeting)
  talk <npc> <message>       — say something directly to the NPC
  ask <npc> about <topic>    — ask a directed question about a topic

EXAMPLES

  talk wuher                      — greet the bartender
  talk wuher What do you know?    — open-ended question
  ask trooper about the patrols   — directed query about patrols

NPC NAMES

  Use the NPC's first name or a unique fragment:
    talk bartender    — matches "Wuher" in Chalmun's Cantina
    talk droid        — if only one droid is present

TOPICS AND WORLD STATE

  NPCs have factions, loyalties, and knowledge tied to the
  current game state. A Republic Soldier will react differently
  to the same question than a Hutt Cartel enforcer.

  NPCs may share rumors, quest hooks, faction intel, or local
  color based on their personality and what they know.

QUEST INTERACTIONS

  Some NPC conversations advance quest chains. Tutorial chain
  NPCs respond to scripted prompts:
    talk Pilot CT-7567 — (Republic Soldier tutorial)
  Smuggler-chain NPCs respond to `give <item> to <npc>` after
  establishing dialogue.

CHEAT SHEET

  talk               — list NPCs in room
  talk <npc> <msg>   — speak to an NPC
  ask <npc> about <topic>  — directed question
