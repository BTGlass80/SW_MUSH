# Per-command help markdown files live here.
#
# Auto-registered command help from cmd.help_text/usage/aliases is a
# baseline — every BaseCommand gets a thin HelpEntry at boot. Drop a
# file here named <key>.md (with slash-encoded keys using __) to
# override that baseline with rich content.
#
# Examples:
#   +sheet.md              for the +sheet command
#   +combat__attack.md     for a future +combat/attack sub-command
#   @dig.md                for the @dig builder command
#
# See data/help/topics/ for examples of the frontmatter + body format
# and HANDOFF_APR17_SESSION47.md for the full spec.
