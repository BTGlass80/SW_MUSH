NOTES F.MAP.1 deliverables and what comes next

This drop ships: static/map_view.js, engine/area_loader.py, data/worlds/clone_wars/maps/mos_eisley.yaml, data/worlds/clone_wars/maps/senate_district.yaml, tools/emit_area_geometry_js.py, this preview, and tests/test_fmap1_area_geometry_loader.py. F.MAP.3: expanded modal — chrome + tier switcher + viewBox animation + ESC handling.

F.MAP.4 / 5: Tier 0 (Site) and Tier 3 (Planet) bespoke renderers.
